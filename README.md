# 🛒 E-Commerce Web Application

Aplicación web e-commerce desarrollada como proyecto fullstack, orientada a la gestión de usuarios, productos y catálogo de ventas.

El sistema fue construido aplicando tecnologías web frontend y backend, integrando base de datos, autenticación y una interfaz responsive utilizando Bootstrap.

---

## 🚀 Funcionalidades

- Registro e inicio de sesión de usuarios
- Gestión de productos
- CRUD de productos
- Catálogo de ventas
- Manejo de precios
- Administración de usuarios
- Conexión con base de datos
- Interfaz responsive

---

## 🛠️ Tecnologías utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
- XAMPP

---

## 🎨 Diseño UI
